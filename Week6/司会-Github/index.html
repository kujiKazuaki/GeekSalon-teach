手順

①「New Repository」⇒githubにアップロードする場所を作る

②$ git remote add origin 「URL」 ⇒どこにアップロードするかを登録する+[git push]を使えるようにする

③$ git push origin master ⇒ 登録したURLにデータをアップロードする
